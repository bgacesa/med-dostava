Med Dostava

Aplikacija koristi Firebase za real-time sinhronizaciju između tebe i dostavljača. Podešavanje traje oko 5 minuta i radi se samo jednom.

1
Idi na console.firebase.google.com i prijavi se sa Google računom
2
Klikni Create a project → ime: med-dostava → Continue (Google Analytics možeš isključiti)
3
U lijevom meniju: Build → Realtime Database → Create database → Start in test mode → Enable
4
Nazad na Project Overview → klikni ikonicu </> (Web app) → registruj app → kopiraj firebaseConfig objekt
5
Unesi podatke ispod — naći ćeš ih u kopiranom firebaseConfig objektu
💡 Dijeljenje sa dostavljačem: Nakon podešavanja, pošalji link aplikacije dostavljaču — oboje vidite iste narudžbe u realnom vremenu. Nema potrebe za registracijom.
